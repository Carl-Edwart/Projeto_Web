Entre Código e Propósito

Este site nasceu como parte de um projeto acadêmico, mas rapidamente ultrapassou a ideia de ser apenas uma atividade escolar. Ele representa um ponto de encontro entre aprendizado, curiosidade e dedicação, reunindo conceitos técnicos e escolhas pessoais que refletem minha trajetória na área de tecnologia.

Durante o desenvolvimento, o foco não esteve apenas no resultado visual, mas principalmente no processo. Cada estrutura em HTML foi pensada para ser clara e semântica, cada detalhe em CSS buscou equilíbrio entre estética e organização, e cada interação em JavaScript teve como objetivo tornar a experiência mais dinâmica e significativa. A separação dos arquivos em pastas específicas segue práticas comuns no mercado, reforçando a importância de um código limpo e de fácil manutenção.

Além da parte técnica, este projeto carrega um valor simbólico. Ele marca uma etapa de evolução acadêmica, na qual conceitos antes abstratos passaram a ganhar forma por meio da prática. O uso de ferramentas como versionamento com Git, arquivos de licença, controle de arquivos ignorados e documentação do projeto demonstra um cuidado que vai além do funcional, aproximando o trabalho de um ambiente profissional real.

Criar este site também foi um exercício de paciência e persistência. Houve ajustes, erros, correções e aprendizados ao longo do caminho, todos fundamentais para o amadurecimento técnico e pessoal. Cada linha de código escrita representa não apenas uma funcionalidade, mas uma escolha consciente de aprender, testar e evoluir.

Por fim, este site não se limita a contar uma história pronta. Ele permanece em constante construção, assim como meu próprio percurso acadêmico. Mais do que um projeto finalizado, ele é um registro do caminho percorrido até aqui e um ponto de partida para tudo o que ainda está por vir.
